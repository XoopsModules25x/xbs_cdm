Class Set

Set object

Holds information about a complete code set. To instantiate this object use:

$setHandler = xoops_getModuleHandler("Set",CDM_DIR);
$setData =& $setHandler->get($id); //where $id is the meta set name

There is no insert() method for this object. This is therefore a read-only object. To write a meta set or code back to the database use the Meta and Code object methods.
